# Advanced GNSS Reliability Analysis Under Signal Degradation

This project presents an advanced simulation and analysis of GNSS positioning performance under various challenging conditions, including:

- Noise
- Multipath interference
- Signal outage
- Urban environments

## Key Features

- Simulation of GNSS degradation scenarios
- Implementation of filtering techniques:
  - Moving Average Filter
  - Initial Kalman Filter
  - Improved Constant Velocity Kalman Filter
- GNSS/INS Sensor Fusion
- Performance evaluation using:
  - RMSE
  - Availability
- Interactive dashboard for visualization

## Results Highlights

- Moving average performs well under smooth noise conditions
- Initial Kalman filter showed poor performance due to model limitations
- Improved Kalman filter significantly enhanced accuracy
- GNSS/INS fusion maintained full availability during outages

- Pandas
